# Digitale Visitenkarte – Acred GmbH

Dieses Projekt wurde im Rahmen meines Praktikums erstellt.

## Ziel

Entwicklung einer modernen digitalen Visitenkarte mit QR-Code-Anbindung.

## Person

Andrei Tihii  
Finanzberater  
Acred GmbH

## Verwendete Technologien

- HTML5
- CSS3
- JavaScript
- vCard
- QR-Code
- Responsive Design

## Verwendete Adobe-Programme

- Adobe InDesign für die Print-Visitenkarte
- Adobe Photoshop für die Bildbearbeitung
- Adobe Illustrator für Logo und Icons

## Funktion

Der QR-Code auf der Rückseite der Visitenkarte führt zu dieser digitalen Kontaktseite.  
Dort kann der Kunde Telefonnummer, E-Mail, Telegram und Firmenadresse sehen und den Kontakt speichern.
